# Domain Glossary

- **Usage window** — a provider-defined allowance measured until a reset boundary.
- **Utilization** — the percentage of a usage window already consumed.
- **Claude 5-hour window** — Claude's short rolling usage window.
- **Claude weekly window** — Claude's longer rolling usage window.
- **Codex weekly window** — Codex's weekly usage window.
- **Unavailable** — no fresh, trustworthy value can be obtained for a provider.
